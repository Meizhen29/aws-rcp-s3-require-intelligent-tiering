## 📋 About This Application

This tool helps you enforce a specific rule on your Amazon S3 buckets. It makes sure your S3 buckets use Intelligent Tiering by applying the right tags. Intelligent Tiering automatically moves data to the best storage class, saving costs without losing access speed.

You do not need coding skills to use this app. It checks your S3 buckets and adds required tags to enable Intelligent Tiering. This helps you manage your cloud storage easily and avoid unnecessary costs.

The tool is built with Amazon Web Services (AWS) best practices. It uses AWS Config and resource control policies to control how S3 buckets are set up. You can use this tool if you manage S3 buckets at your workplace or personal projects.
